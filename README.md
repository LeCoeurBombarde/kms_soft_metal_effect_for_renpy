# kms_soft_metal_effect_for_renpy
Pseudo‑3D Metal Effect

this is a small visual experiment for RenPy in which I created a metallic typographic effect which I call Soft Metal or Pseudo 3d Metal effect. Of course it is not a real 3d render but a pseudo 3d effect that occurs because of smart layers, contrast and subtle shadows. The effect was created solely with standard RenPy capabilities no shaders or external libraries and it is easy to use in other projects. I share this because I am enthusiast about how it looks and perhaps someone else likes it too. It is the result of several coincidences and by no means it was deliberately created. After fiddling for weeks with text shaders I focused on outlines since I could not get even one shader registered. Then when I swapped some colors I suddenly noticed the effect and immediately grabbed it so to speak. I dont know what the values due and wish to keep it this way. The effect looked cool and I dont really care about all the ins and outs, this was an accident that turned out perfect.
